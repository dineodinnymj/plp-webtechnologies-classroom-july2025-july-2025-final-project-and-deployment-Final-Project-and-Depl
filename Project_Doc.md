# Tech Solutions Website

A responsive multipage website built with HTML5, CSS3, and JavaScript for a technology solutions company.

## Features

- **Responsive Design**: Works on all device sizes
- **Multiple Pages**: Home, About, and Contact pages
- **Interactive Elements**: 
  - Mobile navigation menu
  - Testimonial slider
  - Form validation
  - Smooth scrolling
- **Modern UI**: Clean, professional design with animations

## Pages

1. **Home Page** - Hero section, services, testimonials, call-to-action
2. **About Page** - Company story, mission/vision, team members, statistics
3. **Contact Page** - Contact form with validation, contact information, map

## Technologies Used

- HTML5 (Semantic structure)
- CSS3 (Flexbox, Grid, animations, responsive design)
- JavaScript (DOM manipulation, form validation, interactive features)
- Font Awesome (Icons)
